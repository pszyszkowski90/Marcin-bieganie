## Karta 9 — RED FLAGS (kod STOP)

### 🟢 ZIELONY — wszystko OK
Tempo ±5%, nogi/żołądek/głowa ≥6/10, mocz słomkowy, żołądek przyjmuje plan.

### 🟡 ŻÓŁTY — interwencja w następnym okrążeniu
- Tempo zwolnione, lekkie nudności, ból łydki, formujący się pęcherz, mokro/zimno, dół energetyczny → **przygotuj rozwiązanie na następny pit-stop**

### 🔴 CZERWONY — STOP, rozmawiamy
- Wymioty 2× pod rząd, pęcherz duży + ból, ITBS >7/10, dół energetyczny po 14h, nie utrzymuje A-goal → **wspólna decyzja**

### ⚫ CZARNY — DNF, MEDYK (nie pytam Marcina, wzywam)

| Patologia | Objawy → AKCJA |
|---|---|
| **EAH (hiponatremia)** | Dezorientacja, bełkotliwa mowa, zataczanie, obrzęk twarzy/palców, bóle głowy, wymioty, brak moczu mimo picia → **NIE PODAWAĆ WODY** + medyk |
| **Rabdomioliza** | **Mocz koloru coca-coli** = #1 znak. Ból mięśni nieproporcjonalny. Skąpomocz >12h → **SOR/medyk** |
| **Anuria** | Brak moczu >6h → medyk |
| **Hipotermia** | Drży i **PRZESTAJE drżeć** (paradoks!), splątanie, motoryka dłoni → folia NRC + bulion + medyk jeśli 15 min nie pomaga |
| **Dezorientacja** | Nie rozpoznaje mnie, nie wie gdzie jest → STOP + medyk |
| **Krwiomocz brązowy** | = rabdo → SOR |
| **Hipertermia / udar cieplny** | T core >40°C, brak potu w upale → lód pachwiny/szyja + medyk |
| **Ostry ból stawu/ścięgna** z obrzękiem + kulanie 2 okr. | STOP — możliwe złamanie przeciążeniowe |

### Co Paweł pyta przy każdym pit-stopie

```
☐ Mocz — kolor?
  - słomkowy → 🟢 OK
  - ciemny żółty → 🟡 pić więcej
  - brązowy → ⚫ STOP, RABDO
  - brak >6h → ⚫ STOP, ANURIA

☐ Pyt. kontrolne: „Jak masz na imię? Kto wygrał ostatni Mundial?"
  - adekwatna → 🟢
  - wolna ale OK → 🟡
  - dezorientacja → ⚫ STOP

☐ Drżenie / temperatura
  - normalnie → 🟢
  - drży → 🟡 dodaj warstwę
  - PRZESTAŁ drżeć ale zimny → ⚫ HIPOTERMIA

☐ Obrzęk twarzy/palców?
  - brak → 🟢
  - lekki + waga +0% → 🟡 STOP picia
  - wyraźny → ⚫ EAH, STOP
```

### 🚨 BEZ NLPZ (Ibuprom, Ketonal, Nurofen, Diclofenac)
Lipman 2017 RCT: AKI 52% (ibuprofen) vs 34% (placebo). Tylko **paracetamol max 1 g/24h**.
