# Red flags — kryteria zejścia z trasy

> **TL;DR:** Patologie wymagające **bezwzględnego zejścia** + medyka. Marcin i Paweł znają te kryteria **przed startem**. Komunikacja: kolor ⚫ czarny — patrz [[../60-Support/System komunikacji kolory]].

## Bezwzględne kryteria zejścia (consensus 4 LLM)

| Patologia | Objawy | Akcja |
|-----------|--------|-------|
| **EAH (Hiponatremia)** | Dezorientacja, bełkotliwa mowa, zataczanie się, **obrzęk twarzy/palców/kostek**, bóle głowy narastające, wymioty, brak moczu mimo picia | STOP. **NIE podawać czystej wody**. Medyk — bolus sodowy IV / hipertoniczny (3% NaCl). [[EAH hiponatremia\|Pełny protokół]] |
| **Rabdomioliza** | **Brązowy mocz koloru coca-coli** (najważniejszy pojedynczy objaw, wszystkie 4 źródła), ból mięśni **nieproporcjonalny**, skąpomocz >12h | STOP. **Izba przyjęć**. Ryzyko AKI → dializy. [[Rabdomioliza\|Pełny protokół]] |
| **Anuria** | Brak oddawania moczu **>6h** ([GB] kryterium) | STOP → medyk |
| **Hipotermia** | **Drżenie ustające** (paradoks!), splątanie, spowolnienie myślenia, utrata drobnej motoryki dłoni | Przebrać w suche polary + kurtka. **Folia NRC**. Ciepły bulion. Jeśli nie pomaga w 15 min → medyk |
| **Dezorientacja / altered mental status** | Nie rozpoznaje supportera, nie wie gdzie jest | STOP → medyk. **EAH, hipoglikemia, hipertermia, lub udar cieplny** |
| **Krwiomocz** | Brązowo-czerwony = rabdo. Różowy/czerwony = możliwy hematuria wysiłkowa (zwykle łagodna) — konsultacja | Brązowy → STOP. Różowy → konsultacja medyk |
| **Ostry ból stawu/ścięgna z obrzękiem**, zmieniający biomechanikę chodu | Kulanie 2 okrążenia, punktowy ból | STOP — ryzyko **złamania przeciążeniowego** lub zerwania ścięgna |
| **Hipertermia / udar cieplny** | Tcore >40°C, dezorientacja w upale, brak potu mimo wysiłku | STOP. **Cooling agresywny** — lód na pachwiny/szyję, woda na całe ciało. Medyk |

## Dlaczego to KLUCZOWE

W 24h mózg zawodnika **traci sprawność oceny stanu zdrowia**. Marcin po 18h nie potrafi adekwatnie ocenić:
- Czy mocz jest brązowy z rabdo czy z odwodnienia
- Czy zataczanie to zmęczenie czy EAH
- Czy ból kolana to „normalny" czy SR

**Paweł jako kontrola** — patrz [[../60-Support/Paweł — rola operacyjna]] + [[../60-Support/Dziennik supportera]].

## Prewencja najczęstszych problemów

| Problem | Prewencja | Dowody |
|---------|-----------|--------|
| **Pęcherze** | Paper tape przed startem | Lipman 2014 *Clin J Sport Med* RCT — redukcja ~40% **[EBM]** |
| **Pęcherze (kontynuacja)** | Sudocrem między palcami, zmiana skarpet co 6h, Compeed przy pierwszych objawach | [PRAKTYKA] |
| **Odparzenia** | Gruba warstwa Sudocrem przed startem, re-aplikacja co 6–8h | [PRAKTYKA] |
| **Obrzęk stóp** | Para #3 o 0,5–1 rozmiaru większa (Hoka Clifton 10 Wide) | [PRAKTYKA] |
| **Hipotermia nocna** | Ubieraj profilaktycznie o 22:00 (godz. 15 biegu), nie czekaj aż zmarznie | [PRAKTYKA] |
| **EAH** | Sód 500–700 mg/h, ważenie T-6/T-12/T-18, mix izotonik+woda | Hew-Butler 2015 |
| **Rabdomioliza** | NIE NLPZ, hydratacja adekwatna, monitor mocz | Lipman 2017 |
| **Hipertermia** | Cooling proaktywny przy >22°C, sód 800–1000 mg/h | McCubbin 2020 |

## Karta dla Pawła — szybka kontrola

**Co Paweł pyta przy każdym pit-stopie** (patrz [[../60-Support/Dziennik supportera]]):

```
☐ Mocz — kolor?
   - Słomkowy/lekko żółty → 🟢 OK
   - Ciemny żółty → 🟡 Pić więcej
   - Brązowy → ⚫ STOP, RABDO
   - Brak >6h → ⚫ STOP, ANURIA

☐ Stan poznawczy — odpowiedź na pytanie?
   - Adekwatna → 🟢
   - Wolna ale OK → 🟡
   - Dezorientacja → ⚫ STOP

☐ Drżenie / temperatura
   - Brak / w normie → 🟢
   - Drży → 🟡 Dodaj warstwę
   - PRZESTAŁ drżeć ale jest zimny → ⚫ HIPOTERMIA, STOP

☐ Obrzęk twarzy/palców?
   - Brak → 🟢
   - Lekki + waga +0% → 🟡 STOP picia
   - Wyraźny → ⚫ EAH, STOP
```

## NLPZ — bezwzględny zakaz

Patrz [[NLPZ bezwzględny zakaz]]. Lipman 2017 RCT: **AKI 52% (ibuprofen) vs 34% (placebo)**.

## Decyzja: kontynuować czy DNF?

**Po 18h** ([GB] zasada): nie przekonywać do kontynuacji bez fizycznego resetu:
- Posadź Marcina
- Daj rosół
- Masaż łydek
- Świeże skarpety
- **Po 5–15 min sam wstanie lub nie**

**Decyzja DNF zostaje przy Marcinie**, chyba że:
- Spełnia kryteria medyczne (powyższa tabela)
- Wtedy: **Paweł wyzywa medyka**, nie pyta Marcina
