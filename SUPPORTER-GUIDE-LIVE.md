## MONITORING ZDROWOTNY — przy każdym pit-stopie

### 3 pytania obowiązkowe (każde pełne kółko)

```
1. "Jak mocz — kolor?"
   → słomkowy = OK 🟢
   → ciemny żółty = pij więcej 🟡
   → brązowy = STOP, RABDO ⚫
   → brak >6h = STOP, ANURIA ⚫

2. "Żołądek / nogi / głowa — daj 1–10"
   → wszystko ≥6 = 🟢
   → cokolwiek ≤4 = 🟡 interwencja

3. Pytanie kontrolne (od godz. 16): "Jak masz na imię? Ile kółek zrobiłeś?"
   → normalna odpowiedź = 🟢
   → wolna ale poprawna = 🟡
   → dezorientacja = ⚫ STOP
```

## RED FLAGS — ⚫ WZYWAM MEDYKA (NIE PYTAM MARCINA)

| Patologia | Co widzę | Akcja |
|---|---|---|
| **EAH (hiponatremia)** | Dezorientacja, bełkotliwa mowa, obrzęk twarzy/palców, bóle głowy, wymioty, brak moczu mimo picia | STOP. **NIE podawać czystej wody.** Medyk → NaCl 3% IV |
| **Rabdomioliza** | **Mocz koloru coca-coli** | STOP → SOR, ryzyko AKI |
| **Anuria** | Brak moczu >6h | Medyk |
| **Hipotermia** | Drży i **NAGLE PRZESTAŁ drżeć** (paradoks!), splątanie, nie może zapiąć zamka | Folia NRC + bulion + medyk jeśli 15 min bez poprawy |
| **Dezorientacja** | Nie rozpoznaje mnie, nie wie gdzie jest | STOP → medyk |
| **Hipertermia** | T >40°C, brak potu w upale | Lód: pachwiny + szyja + medyk |
| **Ostry ból stawu z obrzękiem** | Kuleje 2 kółka | STOP — złamanie przeciążeniowe |
