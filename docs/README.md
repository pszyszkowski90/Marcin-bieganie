## Jak korzystać z tej bazy

1. **Zaczynasz od pytania** ("dlaczego startujemy 6:30/km, a nie 6:00?") → szukasz odpowiedniej notatki w [[40-Taktyka/Tempo startowe]] lub bezpośrednio w [[80-Decyzje/DEC-002 Tempo startowe konserwatywne]].
2. **Każda decyzja sporna** ma osobny ADR w `80-Decyzje/` — uzasadnienie, alternatywy, dowody.
3. **Każda twarda liczba** ma cytowanie do [[90-Źródła/Literatura kluczowa]].
4. **Aktualizujesz coś?** Edytuj atomową notatkę, dodaj wpis do [[80-Decyzje/Decision Log]] jeśli zmieniasz decyzję.
