# Kluczowy long run T-4

> **TL;DR:** **Sobota ~09.05.2026, 35–40 km, maks 6h, dwufazowy, z elementem nocnym.** Pobudka 2:00 (Marcin śpi 23:00 → optymalne okno nadiru). Pełen protokół żywieniowy 80–90 g CHO/h.

## Werdykt — kiedy i co

**Werdykt:** kluczowy long run w **T-4 sobota (~09.05.2026)**, 35–40 km, **maks 6h**, z **nocnym elementem**:
- **Opcja A:** start 20:00 wieczorem, biegnij do północy
- **Opcja B (preferowana):** **pobudka 2:00**, biegnij 02:00–06:00 — **optymalne okno nadiru** dla 23:00-sleepera (Marcin)

**Źródło:** [PRAKTYKA] Koop 2021, *Training Essentials for Ultrarunning*, 2nd ed.

## Format dwufazowy

| Faza | Dystans | Tempo | % VO2max | Cel |
|------|---------|-------|----------|-----|
| **Faza 1 — natural easy** | Pierwsze 25–30 km | **5:00–5:30/km** (Marcina naturalny easy) | **70–74%** | Aerobowa baza, rozgrzane mięśnie, lekki pre-stress (POWYŻEJ Costa optimum, blisko progu splanchnic ischemia ~70% VO2max) |
| **Faza 2 — race-pace simulation + Costa optimum** | Ostatnie 10–15 km | **5:30–5:55/km** (~71–77% MP velocity) | **60–65%** | DOKŁADNIE [Costa 2017](https://pubmed.ncbi.nlm.nih.gov/28177715/) gut training optimum (60% VO2max @ 90 g CHO/h dał 60–63% redukcję GI distress po 2 tyg.), 24h race pace approximation, durability training |

> **⚠️ Korekta 04.05.2026:** Pierwotna wersja opisywała Fazę 2 jako "race-fueling test pod splanchnic ischemia" — fizjologicznie odwrócone dla sub-3 maratończyka. Faza 2 jest WOLNIEJSZA i mniej intensywna niż Faza 1 dla Marcina. Tempa OK, ramka i etykiety naprawione. Pełne wyjaśnienie: [[../80-Decyzje/DEC-008 Long run dwufazowy]] § Korekta 04.05.2026.

**Decyzja dlaczego dwufazowy** (a nie tylko easy lub tylko race tempo): [[80-Decyzje/DEC-008 Long run dwufazowy]].

## Fueling przez całość

- **80–90 g CHO/h** przez cały long run (**nie celuj 100 g/h** w treningu — testuj górną granicę dopiero w T-3)
- **Dzik Energy Gel 1:0,8** glu:fru (40 g CHO/saszetka) — testowany produkt Marcina, patrz [[30-Żywienie/Produkty testowane]]
- 500–600 ml/h płynów, 500–700 mg Na/h (jak na wyścigu)

## Element nocny — co testujemy

1. **Czołówka** — Petzl Nao RL lub Silva Trail Speed 5XT, baterie zapasowe
2. **Wzrok i koordynacja** w niskim świetle (ostatnia szansa przed startem)
3. **Termoregulacja** — temperatura ~6–10°C, warstwy
4. **Sen post-run** — pobudka 2:00 → bieg 02:00–06:00 → sen 07:00–11:00 (5h spać)
5. **Tolerancja kofeiny w nocy** — testuj protokół 200 mg → 100 mg co 1,5h

## B2B niedziela

**Niedziela ~10.05.2026, 6:00, 15–18 km, tempo 5:30–6:00/km.**

**Cel:**
- Symulacja zmęczonych nóg (5h po long runie)
- Gut tolerance pod fatigue
- Mental — „mogę biegać dalej mimo bólu"

**Wykorzystaj tę okazję:** Paweł przyjeżdża z pełnym wyposażeniem strefy supportu, Marcin biegnie pętlę ~1,7 km × 20+. Trenują 60-sekundowe pit-stopy, dziennik supportera, system kolorów.
